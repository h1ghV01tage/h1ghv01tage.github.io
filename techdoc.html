Installation Guide

Follow these steps to install Galactic Probe on your system:

  1. Download the latest version of Galactic Probe from our official website.
  2. Extract the downloaded archive to your preferred directory.
  3. Open a terminal and navigate to the Galactic Probe directory.
  4. Run the installation script using the following command: $ ./install.sh
  5. Follow the on-screen instructions to complete the installation.
  6. Once installed, Galactic Probe is ready for use.

Usage Instructions

Galactic Probe provides a command-line interface for ease of use. Here are some basic commands to get you started:

  • To perform a basic network scan: $ galacticprobe scan -t target_ip
  • To analyze open ports: $ galacticprobe ports -t target_ip
  • For OS fingerprinting: $ galacticprobe fingerprint -t target_ip
  • Explore additional functionalities in the documentation for advanced use cases.
